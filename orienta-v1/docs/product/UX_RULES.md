# UX RULES

## Regras Oficiais de UX/UI

A interface deve:

- reduzir carga cognitiva;
- evitar excesso de modais;
- seguir fluxo linear;
- manter contexto visivel;
- priorizar status e pendencias;
- possuir tabelas respiraveis;
- padronizar filtros;
- evitar ruido visual.

A plataforma deve parecer:

- moderna;
- institucional;
- leve;
- organizada;
- operacional.
