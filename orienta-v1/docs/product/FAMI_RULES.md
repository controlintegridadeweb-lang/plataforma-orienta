# FAMI RULES

## FAMI

Pontuacao:

- Sim + evidencia aprovada = 1 ponto
- Nao = 0 ponto
- Nao se aplica = excluido do calculo
- Evidencia invalidada = 0 ponto

Formula:
FAMI = pontos obtidos / pontos possiveis

Escala:

- 0% a 20% = FAMI 1
- 21% a 40% = FAMI 2
- 41% a 60% = FAMI 3
- 61% a 80% = FAMI 4
- 81% a 100% = FAMI 5

Niveis:

- geral;
- por eixo;
- por secao.
