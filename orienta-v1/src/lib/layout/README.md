# Layout e design tokens

Tokens de interface compartilhados pela aplicacao logada:

- `design-system.ts` — tipografia, `layout`, `sidebar` (presets de classe).
- `form-surface.ts` — paineis, botoes, tabelas, empty states (`formSurface` / `ds`).
- `admin-page-layout.ts` / `respondent-page-layout.ts` — classes de hero e bleed de pagina.
- `form-fill-route.ts` — deteccao de rota de preenchimento de formulario.
- `staff-module-context.ts` / `respondent-module-context.ts` — contexto de trilha/modulo.

Imagens de hero e URLs de app ficam em `../config/` e `../config/page-assets/`.
